#  Laundry Management System

A **web-based application** built using **ASP.NET Web Forms** and **SQL Server** to streamline the laundry service workflow.  
It enables users to register, place laundry requests, choose services, and track orders online, while providing admin tools for managing customers, clothes, and service pricing.

---

##  Features

###  User Module
- User registration and login  
- Place new laundry requests  
- Select multiple clothes according to service
- Track order and payment status  
- View past and ongoing requests  
- Receive notifications and invoices  

###  Admin Module
- Manage customers and user accounts  
- Add, update, or delete services and clothes  
- Define and manage pricing for each cloth-service type  
- View daily pickups and deliveries  
- Manage notifications and reports  
- Generate invoices and track payments
